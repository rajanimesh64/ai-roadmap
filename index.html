Weekly timeline

How to structure each week for maximum retention

M

Monday — Concepts

Watch lectures, read the theory. No coding yet. Take handwritten notes.

T

Tuesday — Implement

Code the core algorithm from scratch. Resist using high-level wrappers.

W

Wednesday — Experiment

Break things. Change hyperparams, swap architectures, read error messages.

T

Thursday — Paper

Read one seminal paper related to the week's topic. Even 60% comprehension compounds fast.

F

Friday — Project

Complete the week's project. Push to GitHub. Write a one-paragraph summary.

S

Weekend — Review & explore

Catch up on anything missed. Browse arXiv, watch a talk, or start the next week's pre-reading.
