One-time setup

Connect Firebase

This gives every user their own account with progress saved in the cloud — accessible from any device. Takes about 5 minutes.

1

Create a free Firebase project

Go to console.firebase.google.com → New Project → any name → continue.

2

Enable Authentication & Firestore

In the sidebar: Authentication → Sign-in method → Google → Enable. Also enable Email/Password. Then go to Firestore Database → Create database → Start in test mode.

3

Get your config

Go to Project Settings → Your apps → Add web app. Copy the firebaseConfig values and paste them below.

Config is saved in your browser — you only fill this in once.

Weekly timeline

How to structure each week for maximum retention

M

Monday — Concepts

Watch lectures, read the theory. No coding yet. Take handwritten notes.

T

Tuesday — Implement

Code the core algorithm from scratch. Resist using high-level wrappers.

W

Wednesday — Experiment

Break things. Change hyperparams, swap architectures, read error messages.

T

Thursday — Paper

Read one seminal paper related to the week's topic. Even 60% comprehension compounds fast.

F

Friday — Project

Complete the week's project. Push to GitHub. Write a one-paragraph summary.

S

Weekend — Review & explore

Catch up on anything missed. Browse arXiv, watch a talk, or start the next week's pre-reading.
